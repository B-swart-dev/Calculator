# Calculator
A Java 8 project which goal was to create a Calculator Application using JFrame within the Netbeans IDE. With the calculator, users must be able to add, subtract, divide, and multiply numbers. The users should be able to input numbers and math operations using either a computer’s keyboard or clicking on the buttons shown on the interfaces. Just like other basic calculators, your calculator must have M+, M- and MR buttons for adding, removing, and retrieving values from memory, respectively. You are supposed to save the memory values in a “.txt” file.
