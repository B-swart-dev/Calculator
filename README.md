# 🧮 Java Calculator Application

## 📌 About

This project is a **Java 8 desktop calculator application** developed using **Swing (JFrame)** in Apache NetBeans.

Originally built as part of a practical assignment, this project has been refined to demonstrate core Java development concepts including GUI design, event handling, and file-based data persistence.

---

## 🚀 Features

* Basic arithmetic operations:

  * Addition
  * Subtraction
  * Multiplication
  * Division
* Memory functionality:

  * M+ (Add to memory)
  * M- (Subtract from memory)
  * MR (Recall memory)
* Dual input support:

  * Button-based interaction
  * Keyboard input
* Persistent memory storage using a `.txt` file

## ▶️ How to Run

### Option 1: NetBeans

1. Open the project in NetBeans
2. Build and run the project

### Option 2: Command Line

```bash
javac *.java
java Main
```

---

## 📂 Project Structure

* `src/` – Application source code
* `nbproject/` – NetBeans configuration files
* `build/` – Compiled output files

This project demonstrates:

* File handling and persistence
* Structuring a desktop application
